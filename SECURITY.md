# Security Policy

## 🛡 Reporting Security Issues
If you discover a security vulnerability, please **DO NOT** open a public issue. Instead, report it confidentially by sending an email to [ohida2001@gmail.com](mailto:ohida2001@gmail.com).

## 🔒 Supported Versions
| Version | Supported          |
|---------|------------------|
| x.y.z   | ✅ Yes            |
| x.y-1.z | ❌ No (upgrade recommended) |

## 🛠 Fixing Security Issues
- Security patches are prioritized.
- We encourage security-focused contributions.
